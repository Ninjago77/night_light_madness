# Build Guide - Night Light Madness
## Project Hierachy 
 - `hardware/case`: Contains all the 2d AutoCAD (*.dwg*) Design files to laser cut & glue together for making the case & the Acrylic Light Sheet.
 - `hardware/pcb`: Contains all the Electronics Fusion360 (*.f3z* imports to schematics & pcbs) to put on a pcb & attach/solder components & wires to.
 - `software/`: Contains the Arduino Nano's Code to allow the inputs, RTC & Sensors to interact with the LEDs, Display & Buzzer via the Nano.
 - `images/`: Contains all the Images taken of the Project or its components/parts.
 - `README.md`: Provides the introduction, features, usage & overview of the project as well as a gallery of pictures.
 - `BUILDME.md`: Provides the step-by-step guide to recreate the project / customize it for yourself.
 - `List_of_Materials.md`: Provides the components names required (&/or optional) for the making of the project.
