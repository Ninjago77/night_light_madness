## Capabilities

### Here are some of its features, it can be:-
 - Powered with a USB A/C wire
 - Set to a variety of `Modes`, all customizable w/ Code
 - Set to use Any RGB Colour in the Spectrum (or multiple over time)
 - Used to set Alarm(s)
 - Customized with any custom glowing theme
 - Tuned to the light level in the room
 - Enabled to retain time after power loss

## Usage
### It has the following modes (cycle with Pushbutton):-
 - **Mode 1 (White)**: Allows the User to set the Time (Potentiometer) when the Switches are both up (minutes) or when they are both down (hours), opposite directions essentially locks the time.
 - **Mode 2 (~Yellow)**: Allows the User to set a constant Colour of RGBW (Switches) & Brightness (Potentiometer).
 - **Mode 3 (~Magenta)**: Allows the User to set 2 Colours of RGBW to Lerp from (Switches) & the Lerping Speed (Potentiometer).
 - **Mode 4 (~Cyan) `TBD`**: Allows the User to set 3 Colours of RGB/CMY/Random/White to lerp through (Switches), with an exponential Lerping-Speed (Potentiometer)
 - **Mode 5 (Red) `TBD`**: Allows the User to set a constant Colour of RGBW (Switches) but the Brightness will be controlled by the Photoresistor, the sensitivity of which will be controlled by the Potentiometer.
 - **Mode 6 (Green) `TBD`**: Allows the User to set the Alarm Time (Potentiometer) when the Switches are both up (minutes) or when they are both down (hours), opposite directions essentially locks the Alarm time.
 - **Mode 7 (Blue) `TBD`**: Allows the User to set the Auto-Light-Off Time (Potentiometer) when the Switches are both up (minutes) or when they are both down (hours), opposite directions essentially locks the Auto-Light-Off time.
